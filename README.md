# Mobile_Based_Voting_System

MOBILE-BASED VOTING SYSTEM
USER GUIDE
======================================

1. INTRODUCTION
---------------
The Mobile-Based Voting System is a web application used for school elections.
It allows students to vote using their mobile phones or computers. The system
is designed to be easy to use, secure, and reliable.

This guide will help Students, Election Committee members, and Election
Moderators understand how to use the system step by step.

--------------------------------------------------

2. SYSTEM ACCESS
----------------
To open the system:

1. Open a web browser (Google Chrome is recommended).
2. Type the following link in the address bar:

   http://192.168.110.225/Online_Voting/Landing_page.php

3. Press Enter.

You will see the system’s landing page.

--------------------------------------------------

3. STUDENT USER GUIDE
---------------------

3.1 Login
---------
1. Open the system on your phone or computer.
2. Enter your login details.
3. Click "Login".

Note:
Only registered students can log in. If your details are incorrect,
you will not be able to access the system.

--------------------------------------------------

3.2 Vote for a Candidate
------------------------
1. After logging in, you will see the voting page.
2. Look at the list of candidates.
3. Choose the candidate you want.
4. Click "Submit".

Important:
- You can only vote one time.
- You cannot change your vote after submitting.

--------------------------------------------------

3.3 Enter Email
---------------
1. After clicking Submit, a message will appear.
2. Enter your valid email address.
3. Continue to the next step.

Note:
Your voting receipt will be sent to this email.

--------------------------------------------------

3.4 Review Your Vote
--------------------
1. Check your selected candidate.
2. Click "Submit" to confirm.

You will see a message saying your vote has been recorded.

--------------------------------------------------

4. ELECTION COMMITTEE GUIDE
---------------------------

4.1 Login
---------
1. Click the "Login" button at the top-right of the page.
2. Enter the account given by the Election Moderator.
3. Click "Login".

--------------------------------------------------

4.2 Dashboard
-------------
After logging in, you can view:

- Dashboard
- Candidates
- Voters
- Election Results

You can use these pages to monitor the election.

--------------------------------------------------

5. ELECTION MODERATOR GUIDE
---------------------------

5.1 Login
---------
1. Click "Login" on the landing page.
2. Enter your moderator account.
3. Click "Login".

--------------------------------------------------

5.2 Manage Candidates
---------------------
To add a candidate:

1. Go to the Candidates section.
2. Enter all required information.
3. Click "Submit".

Make sure all fields are complete.

--------------------------------------------------

5.3 Manage Voters
-----------------
To upload student voters:

1. Download the student template (Excel file).
2. Fill in the student details.
3. Save the file.
4. Go to "Manage Voters".
5. Click "Choose File".
6. Upload the file.

The system will check if students are allowed to vote.

--------------------------------------------------

5.4 View Results
----------------
- View vote counts in real time.
- Check which candidates are leading.
- View winners and losers for each election year.
- Print the current election results.

--------------------------------------------------

5.5 Download Certificates
-------------------------
1. Go to the Certificates section.
2. Click "Download All Certificates".

Certificates will be created automatically for winners.

--------------------------------------------------

5.6 Start or Stop Election
--------------------------
- Set when voting will start and end.
- You can stop the election anytime by clicking "Stop".

--------------------------------------------------

5.7 Manage Accounts
-------------------
You can:

- Add new accounts
- Edit accounts
- Delete accounts

--------------------------------------------------

6. KEY FEATURES
---------------
- One vote per student
- Easy to use on mobile or computer
- Secure login system
- Real-time results
- Email confirmation after voting
- Upload voters using Excel
- Automatic certificate generation

--------------------------------------------------

7. IMPORTANT NOTES
------------------
- Keep your login details private.
- Check your information before submitting.
- You cannot change your vote after submission.

--------------------------------------------------

END OF GUIDE
